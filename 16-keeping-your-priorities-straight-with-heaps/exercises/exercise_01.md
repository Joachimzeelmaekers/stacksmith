# Exercise 1: Heap Insertion

Implement insertion into a max heap.

## Key Concepts

1. Add element at the end
2. "Trickle up" by swapping with parent if larger
3. Stop when parent is larger or at root

## Complexity

- Time: O(log N)

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter16/exercise01.ts`
- **Go:** `go run src/go/chapter16/exercise01/main.go`
