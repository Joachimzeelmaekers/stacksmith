# Exercise 2: Heap Deletion (Extract Max)

Implement deletion of the root element from a max heap.

## Key Concepts

1. Remove root (the max)
2. Move last element to root
3. "Trickle down" by swapping with larger child

## Complexity

- Time: O(log N)
