# Exercise 2: Trie Insertion

Implement insertion into a trie.

## Key Concepts

- Create nodes for each character if they don't exist
- Mark the final node as end of word

## Complexity

- Time: O(K) where K is the length of the word

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter17/exercise02.ts`
- **Go:** `go run src/go/chapter17/exercise02/main.go`
