# Source Code

This directory contains runnable implementations of all exercises in TypeScript and Go.

## Structure

```
src/
├── typescript/
│   ├── chapter01-20/   # Exercise solutions
│   └── benchmarks/     # Concept benchmarks
└── go/
    ├── chapter01-20/   # Exercise solutions
    └── benchmarks/     # Concept benchmarks
```

## Running Exercises

The easiest way is to use the CLI from the project root:

```bash
yarn start
```

### Manual Execution

**TypeScript:**

```bash
npx tsx src/typescript/chapter09/exercise01.ts
```

**Go:**

```bash
go run src/go/chapter09/exercise01/main.go
```
