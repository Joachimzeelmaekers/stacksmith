# Exercise 5: Array Intersection

Find common elements between two arrays.

## Approaches

- **O(N × M) approach:** Nested loops
- **O(N + M) approach:** Hash set

## Key Insight

Hash set enables O(N + M) intersection instead of O(N × M).
