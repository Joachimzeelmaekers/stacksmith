# Exercise 1: Insertion Sort

Implement and analyze insertion sort - optimized for nearly sorted data.

## Time Complexity

- **Worst case:** O(N²) - reverse sorted
- **Best case:** O(N) - already sorted ← Key advantage!
- **Average:** O(N²)
