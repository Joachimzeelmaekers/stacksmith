# Exercise 2: Reverse a String Using a Stack

Use a stack to reverse a string.

## How It Works

1. Push each character onto the stack
2. Pop each character off the stack to build the reversed string

**Key Insight:** Stacks naturally reverse order due to LIFO (Last In, First Out).
