# Exercise 4: Print Jobs Queue

Implement a print queue that processes print jobs in order.

## Key Concepts

- First job submitted is first to print (FIFO)
- Queue operations: enqueue (add job), dequeue (process job)

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter09/exercise04.ts`
- **Go:** `go run src/go/chapter09/exercise04/main.go`
