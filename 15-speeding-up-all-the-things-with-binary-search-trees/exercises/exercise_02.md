# Exercise 2: BST Search

Implement search in a binary search tree.

## Key Concepts

- If target < current, go left
- If target > current, go right
- If target == current, found!

## Complexity

- Time: O(log N) average, O(N) worst case

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter15/exercise02.ts`
- **Go:** `go run src/go/chapter15/exercise02/main.go`
