# Exercise 4: BST Deletion

Implement deletion in a binary search tree.

## Cases

1. Node has no children: simply remove it
2. Node has one child: replace with child
3. Node has two children: replace with successor (smallest in right subtree)

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter15/exercise04.ts`
- **Go:** `go run src/go/chapter15/exercise04/main.go`
