# Exercise 5: Inorder Traversal

Implement inorder traversal of a BST (visits nodes in sorted order).

## Traversal Order

1. Visit left subtree
2. Visit current node
3. Visit right subtree

## Key Insight

Inorder traversal of a BST produces sorted output.

- **TypeScript:** `npx tsx src/typescript/chapter15/exercise05.ts`
- **Go:** `go run src/go/chapter15/exercise05/main.go`
