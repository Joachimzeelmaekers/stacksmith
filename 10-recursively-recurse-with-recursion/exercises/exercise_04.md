# Exercise 4: Index of X

Use recursion to find the index of 'x' in a string.

## Key Concepts

- Base case: empty string returns -1
- Base case: first char is 'x' returns 0
- Recursive case: 1 + find in rest of string

- **TypeScript:** `npx tsx src/typescript/chapter10/exercise04.ts`
- **Go:** `go run src/go/chapter10/exercise04/main.go`
