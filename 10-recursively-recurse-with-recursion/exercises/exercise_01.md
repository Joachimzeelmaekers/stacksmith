# Exercise 1: Count Characters in Array of Strings

Use recursion to count the total number of characters across an array of strings.

## Key Concepts

- Base case: empty array returns 0
- Recursive case: length of first string + count of rest

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter10/exercise01.ts`
- **Go:** `go run src/go/chapter10/exercise01/main.go`
