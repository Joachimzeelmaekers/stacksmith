# Exercise 3: Triangular Numbers

Calculate the Nth triangular number using recursion.

Triangular numbers: 1, 3, 6, 10, 15, 21, ...
- T(1) = 1
- T(2) = 1 + 2 = 3
- T(3) = 1 + 2 + 3 = 6
- T(N) = 1 + 2 + ... + N
