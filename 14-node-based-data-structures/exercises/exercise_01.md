# Exercise 1: Print All Linked List Elements

Write a function to print all elements in a linked list.

## Key Concepts

- Traverse from head to tail
- Handle empty list case
- Each node has value and next pointer

- **TypeScript:** `npx tsx src/typescript/chapter14/exercise01.ts`
