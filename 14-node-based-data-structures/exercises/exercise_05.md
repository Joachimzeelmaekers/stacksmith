# Exercise 5: Delete Node with Value

Write a function to delete a node with a specific value from a linked list.

## Key Concepts

- Handle deletion of head node
- Find the node before the one to delete
- Update the previous node's next pointer

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter14/exercise05.ts`
- **Go:** `go run src/go/chapter14/exercise05/main.go`
