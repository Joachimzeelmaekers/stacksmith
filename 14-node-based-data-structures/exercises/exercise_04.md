# Exercise 4: Reverse a Linked List

Write a function to reverse a linked list.

## Key Concepts

- Track previous, current, and next nodes
- Update pointers as you traverse
- O(N) time, O(1) space

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter14/exercise04.ts`
- **Go:** `go run src/go/chapter14/exercise04/main.go`
