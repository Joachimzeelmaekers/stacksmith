# Exercise 3: Find Missing Numbers

Given an array of distinct integers, find all missing numbers in the range.

## Key Concepts

- Use quicksort to sort
- Scan for gaps in sequence
- Alternative: use hash set for O(N) solution

- **TypeScript:** `npx tsx src/typescript/chapter13/exercise03.ts`
- **Go:** `go run src/go/chapter13/exercise03/main.go`
