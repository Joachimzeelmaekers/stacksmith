# Exercise 2: Quickselect - Find Kth Smallest

Find the kth smallest element without fully sorting the array.

## Key Concepts

- Uses partitioning like quicksort
- Only recurses into one partition
- Average O(N) time complexity

- **TypeScript:** `npx tsx src/typescript/chapter13/exercise02.ts`
- **Go:** `go run src/go/chapter13/exercise02/main.go`
