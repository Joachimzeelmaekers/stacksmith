# Exercise 2: In-Place Array Reversal

Reverse an array in place using O(1) extra space.

## Key Concepts

- Use two pointers (start and end)
- Swap elements moving inward
- No additional array needed

## Complexity

- Time: O(N)
- Space: O(1)

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter19/exercise02.ts`
- **Go:** `go run src/go/chapter19/exercise02/main.go`
