# Exercise 4: Anagram Generation

Generate all anagrams (permutations) of a string.

## Key Concepts

- For each character, place it first and generate anagrams of the rest
- N! permutations for a string of length N

## Run the Solution

- **TypeScript:** `npx tsx src/typescript/chapter11/exercise04.ts`
- **Go:** `go run src/go/chapter11/exercise04/main.go`
